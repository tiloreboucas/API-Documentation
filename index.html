Sobre

O jQueryCustomSelect (Plugin jQuery Select) tem como principal objetivo facilitar o uso e a customização de selects com uma aparencia mais agradável e mais maleável do que às oferecidas pelos SO/Navegadores, feito utilizando HTML, CSS e JavaScript, tem sua estrutura aceita por todos os navegadores atuais. além de podemos ser submetido com um form ainda pode ser navegável pelo teclado facilitando seu uso.

Exemplos

Inicializando Normalmente

Inicializando jQueryCustomSelect normalmente, à partir do input hidden que existe na página

<!-- jQueryCustomSelect renderizado -->
<div class="customselect_main">
  <div class="customselect_container">
    <input type="hidden" id="SamplejQueryCustomSelect" class="customselect" data-overload="1|Laranja,2|Azul,3|Vermelho,5|Roxo,6|Branco,7|Preto" />
    <div class="customselect_label">Selecione um item</div>
    <button type="button" class="customselect_button"></button>
    <ul class="custonselect_list close">
      <li data-value="1"><span>Novo</span></li>
      <li data-value="2"><span>Usado</span></li>
    </ul>
  </div>
</div>

Alternando entre somente leitura e Leitura e Escrita

É possivel mudar o jQueryCustomSelect para somente leitura utilizando o método .customselect('toReadOnly') ou já inicializa-lo dessa forma usando o atributo data-readonly="true", quando está em modo somente leitura os binds de click, focus e navegação por setas são desabilitados, além de ser adicionado uma classe readonly e um atributo readonly="readonly", para retornar para o modo Escrita e leitura basta utilizar o método .customselect('toReadAndWrite')

Somente Leitura Leitura e Escrita 
<!-- Iniciado normalmente -->
<input type="hidden" id="inputToReadOnly" class="customselect" data-overload="1|Laranja,2|Azul,3|Vermelho,5|Roxo,6|Branco,7|Preto" />
$('#inputToReadOnly').customselect();

<!-- Mudando para somente leitura -->
$('#BtToReadOnly').click(function(e){
  e.preventDefault();
  $('#inputToReadOnly').customselect('toReadOnly');
});

<!-- Mudando para leitura e Escrita -->
$('#BtToReadAndWrite').click(function(e){
  e.preventDefault();
  $('#inputToReadOnly').customselect('toReadAndWrite');
});

Adicionando novos itens

Muitas vezes precisamos atualizar o valor da lista dinamicamente, isso pode ser feito de duas formas, com os métodos .customselect('addNewItem') e .customselect('refresh'). O método .customselect('addNewItem') recebe um objeto JSON com dois atributos, ID e Text {id: '1', text: 'Novo Item'} que são adicionados a lista, e ja o marca como item selecionado, mudando o valor do label e do value do input. Atenção: O Plugin não trata se os dados inseridos são válidos, o método .customselect('refresh') deve ser utilizado quando os valores do atributo data-overload="" são alterados manualmente.

Adicionar 

<!-- Iniciado normalmente -->
<input type="hidden" id="inputToReadOnly" class="customselect" data-overload="1|Laranja,2|Azul,3|Vermelho,5|Roxo,6|Branco,7|Preto" />
<input type="text" id="idAddNewItem" placeholder="Novo ID" />
<input type="text" id="textAddNewItem" placeholder="Insira />
$('#hiddenAddNewItem').customselect();

<!-- Adicionando Itens -->
$('#btAddNewItem').click(function(e){
  e.preventDefault();
  $('#hiddenAddNewItem').customselect('addNewItem', {id: $('#idAddNewItem').val(), text: $('#textAddNewItem').val()});
});

Como usar

HTML

O jQueryCustomSelect é inicializado a partir de um input hidden, esse hidden tem valores que serão usados para popular a lista, os valores devem ser colocados um um atributo data-overload=""

<!-- Input com dados para montar o jQueryCustomSelect -->
<input type="hidden" id="sample" class="customselect" data-overload="1|Novo,2|Usado" />

JavaScript

O jQueryCustomSelect funciona como plugin jQuery, sendo preciso apenas, inicializa-lo

<!-- Função que inicia o jQueryCustomSelect -->
$('#sample').customselect();

Opções

Opções podem ser passadas via data attribute, para isso, esses atributos devem ser declardos no hidden com no nome data-, como data-overload=""

Nome Tipo Padrão Descrição
overload String "" Os itens devem ser separados por virgula( , ), cada item da lista tem value e text, que por sua vez é separado por um vertical pipe( | ) data-overload="1|Novo,2|Usado"
label String "Selecione um item" Texto usado para legenda do select
readonly Boleano false ativa o modo somente leitura do select
ordination String asc A ordenação é feito nos textos do campo e pode ser usados asc(a -> z) ou desc(z -> a)
limit Inteiro 3 Limite de Linhas Exibidas ( Com base na altura da primeira linha ), caso tenham itens com mais de uma linha será resitado a altura da primeira linha

Metodos

.customselect()

Inicializa o CustomSelect

$('#sample').customselect();

.customselect('toReadOnly')

Muda jQueryCustomSelect para modo somente leitura

$('#sample').customselect('toReadOnly');

.customselect('toReadAndWrite')

Muda jQueryCustomSelect para modo Escrita e Leitura

$('#sample').customselect('toReadAndWrite');

.customselect('addNewItem')

Adicona um item ao Select durante a execução dele

$('#sample').customselect('addNewItem', {id: '1', text: 'Novo Item'});

.customselect('refresh')

Durante a execução, você pode mudar os valores do data-overload="" e atualizar a lista

$('#sample').customselect('refresh');

Eventos

onChange

Ao selecionar um item ou mesmo ao navagar pelo teclado a cada troca ele dispara esse evento

$('#sample').customselect({
  onChange: function(){
    // seu código...
  }
});
